# 개요
간단한 쇼핑몰 프로젝트

Use : Spring, JPA, Mysql, Docker, AWS

셀러와 구매자 사이를 중개해 주는 전통적인 커머스 서버를 구축한다.

## 회원 서버
### 공통
 - [ ] 이메일을 통한 인증번호 발송
### 구매자
 - [ ] 회원가입
 - [ ] 예치금 충전
### 셀러
 - [ ] 회원가입
 - [ ] 매출 조회

## 주문 서버
 
### 구매자
 - [ ] 상품 검색
 - [ ] 상품 상세 페이지
 - [ ] 장바구니에 추가 
 - [ ] 장바구니 확인하기
 - [ ] 주문하기
 - [ ] 주문 내역 메일로 발송하기
### 셀러
 - [ ] 상품 등록
 - [ ] 상품 변경
 - [ ] 상품 삭제
 - [ ] 재고 더하기
 - [ ] 재고 뺴기
